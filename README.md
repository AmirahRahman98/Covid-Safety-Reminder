# Covid-Safety-Reminder
Covid Safety Reminder is a safety alert system used to control the spread of contagious diseases. It implies that people physically distance themselves from one another, reducing close contact, and thereby reducing the spread of a contagious disease (such as the COVID-19 Disease). The system would detect the social distancing between people.
